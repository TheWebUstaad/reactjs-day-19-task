Task Requirements

üìå Task Description:

Aap ne ek Single Page Islamic Utility App Banani hai React ke andar, jisme multiple Islamic tools/pages honge. Har page routing ke zariye navigate ho aur component structure follow kare.

üóÇ Required Pages (Total 6 Pages):

1. Qaza Namaz Tracker ‚Äì /qaza-namaz

  • User se poocha jaye: "Aap ne Qaza count kis saal se start karna hai?"
  • Start Year input (eg, 2015), End Year default: current year
  • Har din 6 namazon ka count (Fajr, Zuhr, Asr, Maghrib, Isha,Witr)
  • React se total missed namazon ka auto-count dikhao (years √ó 365 √ó 6)
  • Simple display or table ho with total calculation

2. Zakat Calculator ‚Äì /zakat

  • User se input lo: gold, cash, silver (number fields)
  • Basic calculator banayein ‚Äì amount show karo (formula optional)

3. Tasbeeh Counter ‚Äì /tasbeeh

  • Digital counter with 2 buttons: Increment, Reset
  • State use karke counting update ho

4. Asma Ul Husna ‚Äì /names-of-allah

  • 99 Names show ho grid ya card view mein
  • Data array se ho + map() ka use ho
  • Every name component ko props se mile

5. About Me Page ‚Äì /about

  • Teen sections: Introduction (Intro.jsx), Skills (Skills.jsx), Resume (Resume.jsx)
  • Har section props use kare

6. Islamic Quiz ‚Äì /quiz

  • Islamic MCQs ka chhota quiz app banayein
  • React state aur props ka use ho
  • Har sawal ek component ho aur option choose karne ke baad correct/incorrect batayein

Technical Guidelines

  • Project Name: islamic-utilities-spa
  • Use npx create-react-app ya vite

  • Routing: setup App.jsx mein ho

Sample App.jsx Routing Setup

import { BrowserRouter as Router, Routes, Route } from 'react-router-dom';
import Navbar from './components/Navbar';
import QazaPage from './pages/QazaPage';
import ZakatPage from './pages/ZakatPage';
// ... other imports

function App() {
  return (
    <Router>
      <Navbar />
      <Routes>
        <Route path="/qaza-namaz" element={<QazaPage />} />
        <Route path="/zakat" element={<ZakatPage />} />
        <!-- other routes -->
      </Routes>
    </Router>
  );
}

export default App;
